## Teste você mesmo
1- Clona o projeto
```bash
  git clone https://github.com/Underkyu/Criatil_2.0
```

2-Entre na pasta do projeto

```bash
  cd Criatil_2_0
```
3-Entre na pasta de HTML
```bash
  cd html
```
4-Entre na pasta de HTML
 Procure pelo arquivo "index.html" e o abra.
